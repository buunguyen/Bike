Bike.Random


Helper object used to generate random numbers.

Methods
NameDescription
Bike.Random.next_int( bound1, bound2 )
Generates a random whole number within a specific range.

Parameters
bound1
      Type: Bike.Number
      The minimum value of generated number if bound2 exists. Otherwise, this is the maximum value of the generated number.
bound2
      Type: Bike.Number
      The maximum value of generated number.

Return Value
Type: Bike.Number
A random whole number.
