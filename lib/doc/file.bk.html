Bike.FileSystem.each( path, fn , exts , recursive )
Iterates files and invokes the given function on each file name.

Parameters
path
      Type: Bike.String
      The path of the root folder.
f
      Type: Bike.Function
      A function to be called on each iterated file.
recursive
      Type: Bike.Boolean
      Search files recursively or not. False if not specified.
exts
      Type: Bike.String or Bike.Array
      File extensions. If not specified, all files will match.
